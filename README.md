# EasyCV 简历生成器 (v6.0.0 Stable)

EasyCV 是一款专为极客与工程师量身定制的、100% 纯本地离线运行的单页 A4 简历生成器与排版神器。

## 🌟 核心特色

- **零延迟就地渲染**：完全剥离了繁重的第三方排版框架，采用纯 vanilla JS 与极致性能的 DOM 直刷技术。
- **100% 本地与开源合规**：绝不内置任何侵权的商业闭源字体，支持无网离线直接双击运行，完美保护您的隐私。
- **神级本地字体导入**：内置基于现代浏览器 IndexedDB 的超大容量二进制数据库。您可以秒级导入并永久驻留您本人的任何私人珍藏字体（如锤子工匠、苹方等），突破 5MB 浏览器缓存限制。
- **黄金排版间距配对**：点击不同的字体时，排版引擎会自动联动调节字号、行高、栏间距，带给您物理像素级别的黄金视觉比例。
- **无衬线字体极致美学**：专注于高级工程师、极客和科学排版的现代无衬线（Sans-serif）审美，彻底剔除了冗余的装饰性衬线字体。
